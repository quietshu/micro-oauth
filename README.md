OAuth micro service for GitHub. Deploy via Now: 
```
now quietshu/micro-oauth -e PROVIDER=GitHub -e CALLBACK_URL=https://mak.ink -e CLIENT_ID=16d340c91cc32ce7b321 -e CLIENT_SECRET=XX
```
